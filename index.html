Served

A C++11 library for quickly building RESTful web servers.

Focus on simplicity

Served was designed primarily to create RESTful C++ web services with ease and simplicity.

To demonstrate, here's an example hello world application: 

#include <served/served.hpp>

int main(int argc, char const* argv[]) {
	// Create a multiplexer for handling requests
	served::multiplexer mux;

	// GET /hello
	mux.handle("/hello")
		.get([](served::response & res, const served::request & req) {
			res << "Hello world!";
		});

	// Create the server and run with 10 handler threads.
	served::net::server server("127.0.0.1", "8080", mux);
	server.run(10);

	return (EXIT_SUCCESS);
}

$ curl http://localhost:8080/hello
Hello world!

URL Parameters out of the box

Served supports URL parameters for easy RESTification. 

mux.handle("/users/{id}")
	.get([](served::response & res, const served::request & req) {
		res << "User: " << req.params["id"];
	});

$ curl http://localhost:8080/users/bob
User: bob

You can also specify regex restrictions on those parameters. 

mux.handle("/users/{id:[0-9]+}")
	.get([](served::response & res, const served::request & req) {
		res << "Num id: " << req.params["id"];
	});

mux.handle("/users/{id:[a-zA-Z]+}")
	.get([](served::response & res, const served::request & req) {
		res << "Text id: " << req.params["id"];
	});

$ curl http://localhost:8080/users/1234
Num id: 1234

$ curl http://localhost:8080/users/test
Text id: test
